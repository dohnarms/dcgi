REQUIREMENTS:

This library has no requirements other than the standard libc library.
I build it with GCC and GNU Make, but there is no reason that other
software can't be used (that I know about).


BUILD:

To build the library, simply type "make".  To install the library,
type "make install", where it installs into the directory /usr/local
by default.


DOCUMENTATION:

The manual is the file dcgi_manual.pdf.
